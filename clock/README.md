# Clock

## [Writeup and documentation](https://rabid.audio/projects/synth/clk/)

## Features

- BPM from 15 to 400
- two outputs with relative rates
- 3-digit LED 7-segment display
- swing control
- tap tempo
- CV in
- pause
- save/load/reset settings
